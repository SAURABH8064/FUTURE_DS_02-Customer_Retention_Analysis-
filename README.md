# Customer Retention & Churn Analysis

## Internship
Future Interns – Data Science & Analytics

## Project Overview
This project analyzes customer retention and churn data using Microsoft Excel. The objective is to identify customer churn patterns, understand retention trends, and provide business recommendations to improve customer satisfaction and reduce customer loss.

## Objective
- Analyze customer churn.
- Identify retention trends.
- Find key factors affecting churn.
- Provide actionable business recommendations.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Bar Chart
- Pie Chart
- Line Chart

## Dataset
Telco Customer Churn Dataset

## Excel Analysis
- Data Cleaning
- Customer Churn Analysis
- Contract-wise Analysis
- Tenure Analysis
- Monthly Charges Analysis
- Dashboard Creation using Pivot Tables and Charts

## Dashboard Includes
- Churn Distribution
- Customers by Contract Type
- Tenure Analysis
- Monthly Charges Analysis

## Key Insights
- Month-to-Month contract customers have the highest churn.
- Customers with shorter tenure are more likely to leave.
- Long-term customers have better retention.
- Customers with higher monthly charges show a higher churn rate.

## Recommendations
- Encourage long-term contracts.
- Offer discounts to new customers.
- Improve customer support.
- Introduce loyalty programs.
